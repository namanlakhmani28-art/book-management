# Library Book Management System
## Description
A system software to manage library operations like adding books, issuing and returning books, registering members, and searching the catalog efficiently.
### Features
- Add, update, and delete books
- Register new members
- Issue books to members
- Return books and track overdue books
- Search books by title, author, or category
- View book issue history
## Technology Stack
- Backend: Python
- Database: None
- Tools: Git, Unit Testing
